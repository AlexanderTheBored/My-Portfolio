► GENERAL
help                      List all commands
about                    About this system
clear                    Clear the terminal
exit                     Close this window
► EVENTS
event add <day> <HH:MM> "title"
Add a personal event to the schedule
event list              List all events
event remove <id>       Remove by ID
► NOTES
note "your text"        Pin sticky to desktop
note list              List all notes
note remove <id>       Remove by ID
► WEATHER
wx <state>              Set weather state
clear · cloudy · overcast · fog · rain · storm · night · auto
► ADDING AN EVENT
C:\ROVEL> event add friday 18:00 "Study session"
C:\ROVEL> event add monday 12:30 "Lunch w/ friends"
C:\ROVEL> event list
► PINNING A NOTE
C:\ROVEL> note "Buy coffee before 8AM class"
C:\ROVEL> note list
C:\ROVEL> note remove 12345
► CHANGING WEATHER
C:\ROVEL> wx storm
C:\ROVEL> wx night
C:\ROVEL> wx auto
► TIPS & TRICKS

🌦️  wx auto fetches live weather from Mandaue City
🕐  The schedule widget is live — updates every 30 seconds
⬆️  Use arrow keys in CMD to scroll through command history
🖱️  All windows are draggable and resizable
