MangaView • Details

CLI downloader + JSON metadata + card based UI

Under active refactor
MangaView is split into two parts: scripts that pull manga data, and a static web UI that reads that data and turns it into cards.
  • Helper scripts download chapters from MangaDex into a local assets/manga/ structure.
  • Each series has JSON metadata (title, cover, status, chapter count) that the UI treats as a single source of truth.
  • The frontend is a simple card grid that can later grow into a full reader with filters, tags, and reading progress.
